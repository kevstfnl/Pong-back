# Ecole Ri7
Exercice CI/CD

## Stack
- Express
- Prisma
- bcrypt
- dotenv

## Structure imposée
- `routes/` : Définition des routes par entité
- `controller/` : Logique métier par entité
- `middlewares/` : Middlewares personnalisés

## Installation
```bash
npm install
```

## Lancement
```bash
npm run dev
```

## Variables d'environnement
Créer un fichier `.env` (voir `.env.example`).
